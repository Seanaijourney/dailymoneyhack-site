Privacy Policy

Last updated: April 10, 2026

DailyMoneyHack is committed to protecting your privacy.

1. Information We Collect

Our TikTok application does not collect any personal information from TikTok users. The application only publishes pre-produced videos to our own TikTok account (@dailymoneyhack) using the TikTok Content Posting API.

2. TikTok API Usage

We use the TikTok Content Posting API solely to:

  • Upload pre-produced video files to our own TikTok account
  • Set video titles, descriptions, and privacy settings

We do not access, store, or process any TikTok user data other than our own account credentials.

3. Data Storage

We store only:

  • Our own TikTok API credentials (stored locally and securely)
  • Video files temporarily during the upload process (deleted after upload)

4. Third-Party Services

Our service uses:

  • TikTok Content Posting API (for video publishing)
  • Pexels API (for licensed stock footage)
  • ElevenLabs API (for AI voiceover generation)
  • Anthropic Claude API (for script generation)

5. No User Data Collection

We do not collect, store, or share any personal data from viewers or TikTok users. Our application is a one-way publishing tool only.

6. Children's Privacy

Our content is intended for general audiences. We do not knowingly collect data from children under 13.

7. Changes to This Policy

We may update this policy occasionally. Continued use of our content constitutes acceptance of any changes.
